# Agent Stem

This is my (Jose Viteri) implementation of the task behind the internship application at JetBrains for Fall 2026. Details can be found here: https://internship.jetbrains.com/projects/1792

## Structure

*Writeup*: Can be checked [here](writeup.md).
*Setup*: Can be checked [here](setup.md).
*Measurements*: Can be checked [here](experiments.md).
*Technical reference* (output files, model selection, timing, cost): Can be checked [here](stem_agent/README.md).

Additionally, *stem_agent_build_spec.md* is an agent specification file that can be provided to an agent to generate a similar setup. The zero version is the first draft; the other is the version with improvements made over the course of the week.
